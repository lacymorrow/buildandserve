# Markdown Content

This is a **markdown** example with _formatting_. The floating editor supports full markdown syntax!

## Text Formatting

You can use various text formatting options:

- **Bold text** using `**double asterisks**`
- _Italic text_ using `_underscores_`
- ~~Strikethrough~~ using `~~tildes~~`
- `Inline code` using backticks
