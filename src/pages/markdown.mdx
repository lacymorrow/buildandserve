# ShipKit Pages Layout

## Overview

ShipKit provides a simple layout wrapper for the Pages Router. The `PagesLayout` component adds a consistent header and footer to your pages.

## Usage

```tsx
import { PagesLayout } from "@/components/layouts/pages-layout";

export default function MyPage() {
  return (
    <PagesLayout title="My Page">
      <div>Page content goes here</div>
    </PagesLayout>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | ReactNode | (required) | The page content |
| `title` | string | undefined | Page title (sets document title) |
| `hideHeader` | boolean | false | Whether to hide the header |
| `hideFooter` | boolean | false | Whether to hide the footer |
