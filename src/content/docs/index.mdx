# Welcome to <SiteName />

ShipKit is a modern, opinionated, and feature-rich Next.js website and app starter kit.

## Core Technologies

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Package Manager**: `pnpm` recommended. Also supports `npm`, `bun`, and `yarn`.
- **Authentication**: Auth.js v5 (formerly NextAuth)
- **UI**: Shadcn/UI + Tailwind CSS
- **CMS**: Payload CMS v3
- **Visual Builder**: Builder.io
- **Email**: Resend
- **Database**: PostgreSQL + Drizzle ORM
