# Deployment

ShipKit is optimized for deployment on [Vercel](https://vercel.com), with built-in analytics and monitoring.

## Features

- Zero-config deployment
- Automatic HTTPS
- Edge functions
- Analytics
- CI/CD integration
- Preview deployments

## Configuration

Add these environment variables for deployment features:

```env
# Optional - Vercel integration
VERCEL_ACCESS_TOKEN=your_access_token
VERCEL_CLIENT_ID=your_client_id
VERCEL_CLIENT_SECRET=your_client_secret

# Optional - Analytics
NEXT_PUBLIC_VERCEL_ANALYTICS=true
```

## Quick Deploy

1. Click the "Deploy" button in the README
2. Connect your GitHub repository
3. Configure environment variables
4. Deploy

## Manual Deploy

1. Install Vercel CLI: `npm i -g vercel`
2. Login: `vercel login`
3. Deploy: `vercel deploy`

## Environment Setup

1. Copy `.env.example` to `.env`
2. Configure all required services
3. Add variables to Vercel dashboard
4. Set up preview variables
